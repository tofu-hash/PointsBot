# Points Bot

Бот для записи небольших целей и 
выдачи баллов за их завершение. 


![logo.jpg](source%2Fscreenshots%2Flogo.jpg)

# Запуск

Установи нужные библиотеки:
```bash
pip3 install aiogram
```

Установи `API_KEY` в `config.py`:
```python
# Config.py

API_KEY = '62843...fO70g'
```

Запусти бота (`run.py`):
```bash
cd PointsBot
```
```bash
python3 run.py
```
